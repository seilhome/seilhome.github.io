# 강릉 우미 린 더 프리미어 홈페이지

## 변경사항
- 상담신청 하단 고정 버튼 유지
- 방문희망일: 달력 선택
- 방문희망시간: 시간 선택
- 상담신청 완료 시 고객 화면 안내창 표시
- 구글 스프레드시트 저장 가능
- Gmail 알림용 Apps Script 코드 포함

## 업로드 방법
GitHub 저장소에 아래 항목을 모두 업로드/덮어쓰기하세요.

- index.html

- images 폴더
